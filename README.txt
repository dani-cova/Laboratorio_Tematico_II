
El objetivo de nuestro proyecto es el de crear una aplicacion la cual 
pueda clasificar y permita visualisar los tweets sobre el COVID-19

Antes de intentar correr el programa revisa nuestro manual de usuario

Se necesita tener:
- Haber descargado python
- Descargar el Import-export
- Haber descargado Django
- Haber descargado matplotlib
- Descargar el Proyecto.zip

Requerimientos Mínimos:
OS: Microsoft Windows XP/Vista/7 (32/64bits)
Procesador: 2GHz
Memoria: 1 GB RAM
Gráficos: DirectX 9.0c-compatible graphic card
DirectX: 9.0c
Hard Drive: 3 GB HD space 

El programa fue construido con:
python
D-jango
matplotlib
